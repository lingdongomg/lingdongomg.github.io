剑指 Offer 48. 最长不含重复字符的子字符串

题目

请从字符串中找出一个最长的不包含重复字符的子字符串，计算该最长子字符串的长度。

分析

动态规划

这个解法和我原先的思路最像，并且补上了我没想到的地方。

若计算以下标 n 作为结尾的最大不重复子串， 首先要判断第 n 位的字符有没有出现过。这时就有两种情况 “存在字符重复”，”字符不重复”

字符不重复：最大子串长度 = 上一位的最大子串长度 + 1
f(n) = f(n - 1) + 1

存在字符重复：最大子串长度 = 现在的位置 - 相同字符上一次出现的位置
f(n) = index - lastIndex
该状态转移方程使用场景：
当第 n 位的字符若存在重复且 f(n - 1) + 1 大于 index - lastIndex
因为当前位置最大不重复子串长度必然不可能超过 index - lastIndex，所以此时子串长度就是 index - lastIndex。

同样，f(n - 1) + 1 小于 index - lastIndex时，这个时候说明这对重复字符串中间还存在重复字符串。f(n-1)的意义是以s[n-1]结尾的最长不重复子串，这个子串是没有重复字符的，并且这个字符一定不包含此时的z[n]，因为如果包含的话就会像上面f(n - 1) + 1 大于 index - lastIndex， 所以这里直接把z[n]当作不重复元素增加在上一个子串后，取值f(n - 1) + 1。由此就有了下面的两个不等式：
